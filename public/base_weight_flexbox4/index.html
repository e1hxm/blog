Распределение и выравнивание элементов вдоль осей flexbox

25 December 2024

Свойство justify-content отвечает за распределение флекс-элементов вдоль главной оси. Главная ось является общей для всех элементов, и они расположены на ней, как бусинки на нитке. Это означает, что флекс-элементы взаимодействуют друг с другом вдоль главной оси, влияя на своё положение и отталкивая соседние элементы.

.flex-container {
  display: flex;
  flex-direction: row;
  justify-content: flex-start; /* по умолчанию */

  /*
  justify-content: flex-end;
  justify-content: center;
  justify-content: space-between;
  justify-content: space-around;
  */
}
